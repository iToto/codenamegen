# Codenamegen

Working on a new project but don't have time to think of a quick codename for it? This simple go program will generate a random codename composed of a colour + animal.

## Examples

* KellyGreenWorm
* BottleGreenBee
* VividVioletCheetah
* GreenMunsellPossum

## Usage

```sh
$ go run cmd/codenamegen/main.go
```
